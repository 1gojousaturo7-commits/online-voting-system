# 🗳️ Online Voting System (DCN Micro Project)

## 📌 Project Description
This project is a simple Online Voting System developed using Python socket programming.  
It demonstrates how data is transmitted between a client and server over a network.

---

## 🌐 DCN Concepts Used
- Client-Server Architecture  
- TCP/IP Protocol  
- Socket Programming  
- Data Transmission  
- IP Address and Port Communication  

---

## ⚙️ Features
- User can vote for candidates (A, B, C)  
- One vote per user restriction  
- Server stores and counts votes  
- Multiple clients can connect  

---

## 🛠️ Technologies Used
- Python  
- Socket Programming  
- Replit (Online Execution)  
- GitHub (Project Hosting)
